=== Running the Backend ===
Prerequisite:
- Cloned the backend repo located here: https://github.com/calmakris/cpsc491-snippet-backend

Instructions:
> Open the clone repo mentioned above
> Run the following cmd in terminal: npm run dev

=== Running the Frontend ===
Run these commands in the terminal
> cd 491project
> npm start

# In browser, go to http://localhost:3000
